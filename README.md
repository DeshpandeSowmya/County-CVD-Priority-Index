# County-Level Cardiovascular Prevention Priority Index

This project uses R, SQL, and Tableau to identify U.S. counties where cardiovascular prevention resources may be prioritized based on disease burden, care access, and social conditions.

## Tools
- R (Posit Cloud)
- DuckDB SQL
- Tableau Public
- GitHub

## Data Sources
- CDC PLACES
- County Health Rankings
